# Primo Assignment

## How to run

1. Clone the repository

2. Install the dependencies

   ```bash
   npm install
   ```

3. Run this project with either of the following commands

   ```bash
   npm run build
   ```

   ```bash
   npm run start
   ```

   or run in development mode

   ```bash
   npm run dev
   ```

4. Run the tests

   ```bash
   npm run test
   ```
